# Mouse V1 data download

* Electrophysiological data (in `.nwb`) is available at https://dandiarchive.org/dandiset/000020/.
* Morphological data (in `.swc`) is available at https://download.brainimagelibrary.org/biccn/zeng/pseq/morph/200526/.
* MICrONS EM dataset is available at https://www.microns-explorer.org/cortical-mm3.
